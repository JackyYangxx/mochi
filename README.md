# Desktop Todo

桌面角落里蹲着一只小宠物，提醒你今天还有什么没做完。

点它说话，敲快捷键加待办，到点让 AI 整理成清单推到飞书/钉钉。数据全在本地，API Key 加密存储。

---

## 宠物

透明无边框的悬浮窗，常驻桌面，随时可见。

- **三种状态**：闲置、鼠标悬停、说话中（带脉冲动画）
- **自定义外观**：每个状态都能换上自己的 PNG / JPG / GIF 图片
- **三种尺寸**：小 / 中 / 大，按喜好调整
- **自由拖动**：按住拖到任意位置，下次启动还在那里
- **提示气泡**：闲置时显示通知（如"日报已生成"）

## 添加待办

三种方式随时记录：

- **点宠物** —— 弹出输入框
- **全局快捷键** `Cmd/Ctrl+Shift+T` —— 在任何应用中都能呼出
- **按住语音** —— 说完自动转文字

输入框支持 `Enter` 提交、`Shift+Enter` 换行、`Esc` 关闭。

## 待办管理

- 增删改查、标记完成、搜索
- **子待办**：任务下挂子任务，删父项自动删子项
- **自动排序**：未完成的在上，完成的沉底
- **完成动画**：勾选时粒子特效
- **详情备注**：每项可写备注，`Cmd/Ctrl+Enter` 保存

## 每日提醒

设置提醒时间（支持多个），到点自动：

1. 读取当前未完成待办
2. 调 AI 生成摘要和下一步建议
3. 通过你配置的 IM 工具推到手机

## 日报

每天自动或手动生成工作日报（Markdown），汇总：

- 已完成事项
- 未完成事项
- AI 总结

已完成待办自动归档，保持列表清爽。

## 设置面板

顶部 4 个 Tab：

| Tab | 内容 |
|-----|------|
| 外观 | 宠物大小、三种状态的自定义图片 |
| 智能 | AI 服务地址、模型、API Key（加密存储） |
| 通知 | 提醒时间、IM 推送工具、日报保存目录 |
| 系统 | 开机自启 |

托盘右键菜单可快速打开设置。

## 安装运行

```bash
pnpm install

# Apple Silicon Mac 需重建 SQLite 模块
npx @electron/rebuild -f -w better-sqlite3

pnpm build && pnpm start
```

启动后桌面右下角出现宠物。试用：点宠物 → 写条待办 → `Cmd/Ctrl+Shift+T` 试试快捷键。

## 打包

```bash
# macOS
pnpm build

# Windows x64
pnpm build && pnpm electron-builder --win --x64 --dir
cd release && zip -r Desktop-Todo-vX.X.X-win-x64.zip win-unpacked
```
