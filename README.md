## 桌面宠物

Mochi 以透明无边框窗口悬浮在桌面，作为常驻伴侣。

- **三种状态**：闲置、鼠标悬停、说话中（带脉冲动画）
- **自定义外观**：每个状态支持自定义 PNG / JPG / GIF
- **三种尺寸**：小 / 中 / 大
- **自由拖动**：位置自动持久化
- **提示气泡**：闲置时显示通知

## 添加待办

三种方式覆盖鼠标、键盘、语音：

- **点击宠物** —— 弹出输入框
- **全局快捷键 `Cmd/Ctrl+Shift+T`** —— 在任何应用中都能呼出
- **按住语音** —— 说完自动转文字

输入框：`Enter` 提交，`Shift+Enter` 换行，`Esc` 关闭。

## 待办管理

- 增删改查、标记完成、搜索
- **子待办**：任务下挂子任务，删除父项自动级联删除子项
- **自动排序**：未完成在上，完成沉底
- **完成动画**：勾选时粒子特效
- **详情备注**：每项可写备注，`Cmd/Ctrl+Enter` 保存

## 每日提醒

支持设置多个提醒时间，到点自动：

1. 读取当前未完成待办
2. 调用 AI 生成摘要和下一步建议
3. 通过配置的 IM CLI 工具（钉钉 / 飞书 / Slack 等）推送

## 日报

每天自动或手动生成 Markdown 工作日报，汇总：

- 已完成事项
- 未完成事项
- AI 总结

完成后自动归档已完成待办。

## 本地知识库

为 AI 提供个人上下文，让提醒和日报更贴合你的工作。

- 索引本地 Markdown 笔记（最多 5 个目录 / 30 个文件）
- **Wiki 目录**：Mochi 自身的笔记存放处
- **角色模板**（`role.md`）：填写身份、目标、原则，让 AI 更懂你
- 智能分类与日报生成会参考已索引的笔记内容
